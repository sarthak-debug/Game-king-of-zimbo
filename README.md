# King of Zimbo 🎮

A 2D game made using Godot Engine.

## 🎯 Features
- Smooth player movement
- Custom assets
- Level system

## 🕹 Controls
- Arrow keys / WASD to move

## 🚀 How to run
Open the project in Godot and click Run.
